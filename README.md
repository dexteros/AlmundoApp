# Almundo App

Aplicación desarrollada en React Native para consulta de hoteles

##### Servicio Web: https://almundo-ws.herokuapp.com/

## Instalación


```bash
git clone https://github.com/dexteros/AlmundoApp.git
cd /AlmundoApp
npm install
react-native link
```
